# Portfolio-Khouloud-CHARNI

Projet React + Vite prêt pour VS Code, GitHub et Vercel.

## Lancer en local

```bash
npm install
npm run dev
```

Puis ouvrir l'adresse indiquée, généralement :

```bash
http://localhost:5173
```

## Déployer sur Vercel

1. Envoyer ce dossier sur GitHub.
2. Dans Vercel : Add New Project.
3. Sélectionner le dépôt GitHub.
4. Framework : Vite.
5. Build command : `npm run build`.
6. Output directory : `dist`.
7. Deploy.

## Important

Ce projet n'utilise pas Tailwind ni PostCSS pour éviter les erreurs rencontrées précédemment.
Ne modifie pas `package.json` à la main si ce n'est pas nécessaire.
