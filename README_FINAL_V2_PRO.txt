INSTALLATION :
1. Dézipper ce dossier.
2. Remplacer ton projet actuel par ce dossier, ou remplacer au minimum src/main.jsx et src/styles.css.
3. Vérifier que les assets restent bien dans public/assets.
4. Lancer :

npm install
npm run build
git add .
git commit -m "Ajout version professionnelle et lien univers personnel"
git push origin main

Vercel redéploiera automatiquement.
